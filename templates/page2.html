About This Useless App

Because sometimes learning Flask means building things that don’t matter.

This application was built as part of a CIS class project to practice:

  • Using Flask routes and templates
  • Extending a shared base.html layout
  • Handling form submissions and POST requests
  • Adding a consistent visual theme across multiple pages

Under the hood, the app randomly selects a winning box index (0–8 for a 3×3 grid) and compares it to the value you clicked.
